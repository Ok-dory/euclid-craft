# 유클리드 크래프트

## 개요
유클리드 원론 1권의 공리·공준을 조합(크래프팅)해서 정리를 발견하는 수학 웹게임.
공리/정리가 캐릭터로 의인화되어 등장.

## 프로젝트 플랜
전체 기능 범위, 페이지 구조, 기술 결정, 마일스톤은 docs/plan.md 참조.
새 기능 추가나 구조 변경 시 plan.md와 대조하여 범위 내인지 확인.
plan.md에 없는 기능을 임의로 추가하지 않기.

## 기술 스택
Next.js 15 / TypeScript / Tailwind / shadcn/ui / Zustand / Framer Motion

## 디자인 톤
다크 배경, 골드 포인트 컬러, 기하학적 요소.
Variant 원본 없음 — 아래 토큰 기준으로 일관되게 구현:
- 배경: #0a0a0a ~ #1a1a2e
- 포인트: #c9a84c (골드)
- 텍스트: #ffffff, #a0a0a0
- 카드 배경: #1e1e2e
- 보더: #2a2a3e

## 데이터
- src/data/cards.json — 카드 원본 데이터. 수정 금지.
- src/data/recipes.json — 조합 레시피 원본. 수정 금지.
- 데이터 변경 필요시 나에게 보고.

## 작업 규칙
- plan.md 마일스톤 순서대로 진행
- 임의로 기능 추가하지 않기
- 디자인 토큰 일관성 유지
